- New Logo
- New Icon
- Added some custom easter eggs
- Stage Editor (made by @magnumsrtisswag)
- New fps counter (by @BeastlyGhost)
- Fixed null error in the chart editor

- Judgement counter!
- Winning Icons (from Mic up' Engine)

- Fucked difficulty
